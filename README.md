# 🏋️ GenFit – AI Powered Personalized Fitness Platform

## 🚀 About GenFit

GenFit is an AI-powered personalized fitness platform that combines AI assistance, trainer support, workout tracking, and nutrition analysis to help users achieve better fitness results.

Unlike generic fitness apps, GenFit focuses on:
- Personalized workout plans
- AI-powered food analysis
- Trainer consultation
- Progressive overload tracking
- Real-time fitness guidance

---

## ✨ Features

### 🤖 AI Fitness Assistant
- Answers fitness questions
- Provides workout guidance
- Gives nutrition insights

### 🍱 AI Food Analysis
- Upload food images
- Detect calories, proteins, carbs, and fats instantly

### 💬 Trainer Booking & Chat
- Hire trainers hourly
- Live chat support
- Personalized guidance

### 📈 Workout Logging
- Track sets, reps, and weights
- Monitor fitness progress effectively
